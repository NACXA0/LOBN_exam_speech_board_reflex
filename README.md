# LOBN_exam_speech_board_reflex
鲁班讲题白板

## 技术栈
- reflex
- reflex 按需包裹 npm 组件

## 场景
- 内网使用，无需密码
- 讲题主页，类似于 ppt
- 后台集成文件管理与文件编辑
- 基于本地文件，无需数据库

## 页面
1. **选择题库** - 选择后开始
   - 方便操作，点击后快速进入
2. **讲题主页**
   - 只有选择题
   - 界面样式：题干、选项、折叠的答案与解析、翻页按钮
   - 功能：点击选项之后。1. 正确选项底色变绿。2. 展开解析
3. **后台** - 上传题库
   - 支持离线上传、文件上传、在线上传
   - 离线上传：有自动解析文件的 py 脚本，直接将处理好的文件放到对应的路径就可以
   - 文件上传：docs 或者 txt 上传后自动进行解析->然后可编辑的预览->确认后保存并删除原始文档
   - 在线上传：文本框的方式录入文本->实时生成可编辑的预览->确认后保存并删除原始文档

## 题库存储方式
- JSON 或者 MD。图片用 base64 嵌入到文本中，展示时解析出来。
- 一开始的选择题库就是选择使用哪个 JSON 或者 MD 文件

## 运行方式
```bash
cd LOBN_exam_speech_board_reflex
reflex run
```

访问：http://localhost:3000
- `/` - 选择题库
- `/workspace` - 讲题主页  
- `/admin` - 后台管理
